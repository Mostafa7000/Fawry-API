# Fawry-API
 A web API for a Fawry-like system

## Development enviroment
- Java 17
- Maven
- SQLite database
- Intellij IDE (Can Run on Eclipse fine)

## dependencies
- Spring web
- Lombok
- SQLite JDBC

## How to run the server
1. In Eclipse, import the project as a maven project, And run the FawryApplication class
